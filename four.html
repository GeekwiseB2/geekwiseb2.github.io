Practice

  • Create a new Github repository
  • Clone to your local machine
  • Set up your project on master branch and push once
  • Create a new branch
  • On your new branch, create a test site with:
    • Nav
    • Jumbotron
    • Button(s)
    • Well(s) with some content
    • Basic styling
  • Create pull request
  • Slack me the link to your pull request
  • When I give the ok, merge and delete your branch (both local and github.com)

Take Home
